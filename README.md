# ✨ Neonotex

> **Think It. Push It.**

Neonotex is a modern all-in-one productivity platform designed for students, creators, and professionals. It combines note-taking, AI assistance, productivity tools, and smart utilities into one beautiful workspace.

🌐 **Live Demo:** https://www.neonotex.space

---

## 🚀 Features

### 📝 Notes
- Create, edit, and organize notes
- Rich text editor
- Auto-save
- Collections
- Favorites
- Trash management

### 🤖 NeoBot AI
- Grammar correction
- Rewrite sentences
- Improve writing
- Summarize notes
- Explain text
- AI writing companion

### ✅ Tasks
- Create and manage tasks
- Due dates
- Reminders
- Task completion tracking

### 📱 Neo QR
Create beautiful QR codes instantly.

Supports:
- URL
- Text
- Email
- Phone
- Contact
- Wi-Fi
- Location

Features:
- Multiple QR styles
- Custom colors
- Download PNG
- Copy QR
- Adjustable QR size

### 🧾 Invoice Generator
Create professional invoices in seconds.

- Business details
- Client details
- Invoice items
- Auto calculations
- PDF export
- Multiple templates

### 🛍️ Neo Store
Purchase exclusive Neonotex products.

(Current Product)
- ✏️ Neon Premium Pencil

### 🎮 Neo Play *(Coming Soon)*
Fun educational games including:
- Word Puzzle
- Word Scramble
- Word Search
- Memory Game
- Typing Challenge
- Brain Quiz

## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/yourusername/neonotex.git
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build project

```bash
npm run build
```
